# Receive_LoRa_RAK3172_STM32IDE
Projeto de firmware para STM32 desenvolvido no STM32CubeIDE que implementa a recepção de pacotes via rádio LoRa utilizando o módulo RAK3172. O código configura o rádio em modo LoRa (ponto-a-ponto), trata eventos de recepção, extrai campos do payload e exibe informações como RSSI, SNR e dados recebidos via interface serial para depuração.
